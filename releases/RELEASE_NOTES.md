## 📋 System Requirements

- Windows 10/11
- PowerShell (included with Windows)
- AudioDeviceCmdlets PowerShell module (recommended)

### Installing AudioDeviceCmdlets
```powershell
# Run PowerShell as Administrator
Install-Module AudioDeviceCmdlets -Force
```
